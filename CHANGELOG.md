# Changelog

All notable changes to MomCare will be documented in this file.

## [1.0.0] - 2026-06-16

### Added
- Initial release of MomCare app
- Daily Planner with daily, weekly, monthly views
- Smart Shopping List with categories
- Family Memory Vault for precious moments
- Household Expense Tracker with charts
- Comprehensive Teacher Toolkit
- Personal section "From Aadityaa"
- Dark and light theme support
- Offline-first functionality
- SQLite local database
- Push notifications
- Voice input for tasks
- Biometric authentication
- Beautiful Material Design 3 UI
- Multi-language support ready
